## AI Finance Assistant

## Project Description
AI Finance Assistant is a Python-based personal finance management system that helps users manage income, expenses, budgets, and financial reports. The application uses SQLite for data storage and Matplotlib for charts and visualizations.

## Features
User Registration and Login
Add Income and Expenses
View Transactions
Update and Delete Transactions
Budget Management
Budget Warning System
Monthly Financial Reports
Expense Pie Chart
Income vs Expense Chart
CSV Report Export
AI-based Financial Suggestions

## Technologies Used
Python
SQLite
Matplotlib

## How to Run the Project
1. Install Required Libraries
pip install -r requirements.txt

2. Run the CLI program
python main.py

3. Run the web frontend
python app.py

Then open http://127.0.0.1:5000 in your browser.
